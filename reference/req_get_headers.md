# Get request headers

Retrieve custom headers set on the request. Use
[`req_dry_run()`](https://httr2.r-lib.org/reference/req_dry_run.md) to
get all headers, including those automatically generated by curl.

## Usage

``` r
req_get_headers(req, redacted = c("drop", "redact", "reveal"))
```

## Arguments

- req:

  A httr2 [request](https://httr2.r-lib.org/reference/request.md)
  object.

- redacted:

  What to do with redacted headers?

  - `"drop"` (the default) drops them.

  - `"redact"` replaces them with `<REDACTED>`.

  - `"reveal"` leaves them in place.

## Value

A named list.

## Examples

``` r
req <- request("http://example.com")
req <- req_headers(req, a = 1L, b = 2L, .redact = "a")

req_get_headers(req, "drop")
#> $b
#> [1] "2"
#> 
req_get_headers(req, "redact")
#> $a
#> [1] "<REDACTED>"
#> 
#> $b
#> [1] "2"
#> 
req_get_headers(req, "reveal")
#> $a
#> [1] "1"
#> 
#> $b
#> [1] "2"
#> 
```
