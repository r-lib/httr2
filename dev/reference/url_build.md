# Build a string from a URL object

This is the inverse of
[`url_parse()`](https://httr2.r-lib.org/dev/reference/url_parse.md),
taking a parsed URL object and turning it back into a string.

## Usage

``` r
url_build(url)
```

## Arguments

- url:

  An URL object created by
  [url_parse](https://httr2.r-lib.org/dev/reference/url_parse.md).
